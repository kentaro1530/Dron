    {\footnotesize
      \begin{enumerate}
      \item $K^- + d \rightarrow \Sigma_{forward}^{\pm} + \pi^{\mp} + "n"$
      \item $K^- + d \rightarrow n_{detect} + K^0 + "n"$
      \item $K^- + d \rightarrow n_{detect} + \pi^{\mp} + \Sigma^{\pm}$
      \end{enumerate}
    }
    { \scriptsize
      $K^0$ \& $\Sigma_{forward}^{\pm}$ were identified by invariant mass fitting of $\pi^+ \pi^-$ and $n \pi^{\pm}$.\\
      Backward $\Sigma^{\pm}\pi^{\mp}$ were separate by $d(K^-, n \pi^{\mp})"\Sigma^{\pm}"$ fitting, \\which was performed bin-by-bin of $d(K^-, n)"X"$\\
    }
    \vspace{2mm}
    2 step like reaction was observed in $d(K^-, n K^0)"n"$ events.\\
    $d(K^-, n)"\pi^{\mp}\Sigma^{\pm}$ strength was observed in the below $\bar{K}N$ threshold.
